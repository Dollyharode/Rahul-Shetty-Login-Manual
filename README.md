# Rahul Shetty Login - Manual Testing Project

## Project Overview
This repository contains the complete manual testing documentation for the Rahul Shetty Login application. The project demonstrates the complete Software Testing Life Cycle (STLC), including test planning, test case creation, bug reporting, and execution evidence.

## Project Objective
- Verify the functionality of the Login module.
- Validate positive and negative login scenarios.
- Identify and document defects.
- Ensure the application behaves according to the expected requirements.
